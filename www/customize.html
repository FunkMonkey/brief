Customizing Brief's look

You can customize Brief's look using CSS stylesheets, just like a normal webpage. Use the Display tab in Options to select the CSS file.

The easiest way to create your own style is to take the default style and start from there. It's got some useful comments which should help you get going. The structure of the page looks like this:

<body>
    <div id="container">

        <a id="header">
            <img id="feed-image"/>
            <div id="feed-title"/>
            <div id="feed-subtitle"/>
            <div id="clear"/>
        </a>

        <div id="feed-content">

            <!-- Articles go here. Each of them has the following structure: -->
            <div class="article-container">
                <div class="article-header">

                    <div class="expand-button"/>

                    <div class="star-container">
                        <xul:image class="star-image"/>
                    </div>

                    <div class="article-controls">
                        <xul:button class="mark-read"/>
                        <xul:button class="delete-entry"/>
                        <xul:button class="restore-entry"/>
                    </div>

                    <a class="article-title-link"/>
                </div>

                <div class="article-subheader">
                    <div class="article-date"/>
                    <div class="article-authors"/>
                    <div class="feed-name"/>
                </div>

                <div class="article-content"/>
            </div>

        </div>

    </div>
</body>

There are also some common style rules which are additional applied and set the images of the buttons. They are separated, so that they don't clutter the default style, because you probably won't need to change those. You can still overwrite them using the !important rule if you want.

I plan to put up a gallery of users' styles, so once you create one yourself, please let me know!
