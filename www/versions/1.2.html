What's new in Brief 1.2?

New features and improvements:

  • Integration with the new bookmarks system in Firefox 3.
  • Search bar matches only full words and provides Google-like syntax, for example: mozilla OR "firefox extension" -tabs.
  • Improved performance. Checking feeds for updates doesn't lock up Firefox and displaying feeds with a lot of items is snappier.
  • Better keyboard shortcuts, current item can be highlighted automatically.
  • Option to sort oldest items first in unread view.
  • New, easier way of customizing appearance of feeds.
  • A feed-specific setting not to mark changed items back as unread.
  • Tweaks to the default style of feeds.

Fixed bugs:

  • Brief works better with NoScript extension.
  • Improved importing OPML, now Brief handles more kinds of files.
  • After opening an item in unread view and then going back, marking items as read doesn't work anymore
  • Sometimes feeds are skipped when updating multiple feeds.
  • Favicons in the list of feeds sometimes are mixed up.
