What's new in Brief 1.0?

New features and improvements:

  • Buttons on the main toolbar are rearranged.
  • Item date is displayed in a more friendly format using Today, Yesterday, and days of the week.
  • There is label indicating when an item was changed (for example when a blog post was edited). Such item is also marked back as unread again.
  • Next/Previous buttons were added to the feed properties window.
  • Visual changes were to the default style.
  • Headlines view was revamped.

Fixed bugs:

  • Brief's favicon sometimes isn't shown on the tab.
  • Scroll postion and selected entry aren't remembered when refreshing the view.
  • Sorting, removing, or adding many Live Bookmarks at once aren't reflected in the sidebar.
  • Regression: sidebar has too big minimum width.
  • Brief sometimes causes Firefox take a very long time to shut down. Fixed by disabling automatic compacting of the database. User may now choose to compact it when emptying Trash folder.
  • List of shortcut keys is cut off if window is too small.
