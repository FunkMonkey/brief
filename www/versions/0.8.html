What's new in Brief 0.8?


  • See shortcut keys
    Support for keyboard navigation and other shortcuts - you can press ? to see the list of shortcuts at any time.
  • Searchbar now searches for separate words instead of the whole string.
  • When newer versions of entries are available (e.g. when blog posts are edited), Brief updates the entries and marks them as unread again.
  • Pressing Escape clears the searchbox and ends the search.
  • Improve the way the Live Bookmars folder is chosen on the first run, by allowing to select it straight from the sidebar.

Fixed bugs:

  • Toolbar button doesn't stay pressed when opening Brief in a new tab.
  • Entries are sometimes randomly duplicated when updating feeds.
  • Added a hidden preference extensions.brief.database.disableVacuum to disable compacting of the database (for people with extremely large amounts of entries compacting caused Firefox to take ages to shut down).
  • Other minor bugfixes.

You can customize Brief's look. Create your own style and share it!

Navigation
K or Space Select next entry
J or Backspace Select previous entry
Alt + K Show next page
Alt + J Show previous page
Enter Open entry
Ctrl + Enter Open entry in new tab
N Unfold or fold selected entry (when in headlines view)
Entries
M Mark selected entry as read or unread
B Star or unstar selected entry
T Move selected entry to Trash
R Restore selected entry (when in Trash)
Ctrl + M Mark all viewed entries as read
Tab Deselect any entries
View
A Show all entries
S Show starred entries
D Show unread entries
F Toggle headlines view
/ Focus searchbox
? Display list of shortcuts keys
