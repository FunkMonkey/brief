How to subscribe to a feed using Live Bookmarks?

When a website provides a feed, an orange icon is shown in the Location Bar. You can click on it to preview the feed.

Make sure Live Bookmarks is selected as the subscription method and click Subscribe Now.

Pick the folder where you wish to save the Live Bookmark.
