Brief is a light-weight feed reading extension for Firefox. Designed to be easy to use and streamlined, with exactly the right set of features.

What's new in Brief 1.0?

  • Rearranged buttons on the main toolbar.
  • Item date displayed in a more friendly format using Today, Yesterday, days of the week etc.
  • There is label indicating when an item was changed (for example when a blog post was edited). Such item is also marked back as unread again.
  • Added Next/Previous buttons to the feed properties window.
  • Visual changes to the default style.
  • Revamped headlines view.
  • Some bugs fixed.

Release notes

  • NoScript users should add file:// to the list of trusted sites in order for Brief to work correctly
  • On Mac OS X options window may be partially broken
  • Remember that for Brief to work you should bookmark your feeds as Live Bookmarks (not regular bookmarks)
  • Supported languages: bg-BG, da-DK, de-DE, de-DE, el-GR, en-GB, fr-FR, hu-HU, it-IT, ja-JP, nb-NO, nl-NL, pt-BR, ru-RU, sk-SK, tr-TR, zh-CN, zh-TW.
