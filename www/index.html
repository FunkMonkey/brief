What's new in Brief 0.8

  • Support for keyboard navigation and other shortcuts
  • Searchbar now searches for separate words instead of the whole string
  • When newer versions of entries are available (e.g. when blog posts are edited), Brief updates the entries and marks them as unread again
  • Pressing Escape clears the searchbox and ends the search
  • Improve the way the Live Bookmarks folder is chosen on the first run, by allowing to select it straight from the sidebar
  • Fixed a bug causing entries to be duplicated sometimes when updating feeds

Feedback

For feedback, feature requests, and bug reports please use Brief's topic on MozillaZine forums or the discussion board on addons.mozilla.org; I try to follow them closely.

You can also drop me an e-mail at ancestor (at) o2.pl.

Known issues

  • NoScript users should add file:// to the list of trusted sites in order for Brief to work correctly
  • On Mac OS X options window is broken
  • This is not really an "issue" but remember that the feeds you have in your bookmarks must be Live Bookmarks (not feed addresses saved as regular bookmarks) for Brief see them
