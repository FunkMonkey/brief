Brief is a light-weight feed reading extension for Firefox. Designed to be easy to use and streamlined, with exactly the right set of features.

What's new in Brief 1.1?

  • Brief is now compatible with Firefox 3 Beta 1 and 2.
  • Support for exporting and importing OPML (a format for exchanging feeds). Thanks to Christopher Finke, the author of OPML Support extension.
  • Option to automatically mark items as read when the are viewed.
  • The control to show/hide the list of feeds was moved to the main toolbar for better discoverability.
  • Mac OS X: fixed some issues with the Options window and various visual glitches.

Release notes

  • NoScript users should add file:// to the list of trusted sites in order for Brief to work correctly
  • Remember that for Brief to work you should bookmark your feeds as Live Bookmarks (not regular bookmarks)
  • Supported languages: bg-BG, da-DK, de-DE, de-DE, el-GR, en-GB, fr-FR, hu-HU, it-IT, ja-JP, nb-NO, nl-NL, ru-RU, sk-SK, tr-TR, zh-CN, zh-TW.
