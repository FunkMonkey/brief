Brief is a light-weight feed reading extension for Firefox. Designed to be easy to use and streamlined, with exactly the right set of features. Learn more...

What's new in Brief 0.6

  • Ability to limit number of stored entries
  • Preference to always open entry links in new tabs
  • Context menu option to delete all entries from a feed or a folder
  • Ability to hide the sidebar
  • Favicon on the Brief's tab
  • When viewing many feeds at once, labels are shown telling from which feed each entry comes
  • Tooltips for the toolbar button and the status icon which showing detailed information about the unread feeds
  • Plenty of important bugfixes

Known issues

  • NoScript users should add file:// to the list of trusted sites in order for Brief to work correctly
  • On Mac OS X options window is broken
  • This is not really an "issue" but remember that the feeds you have in your bookmarks must be Live Bookmarks (not feed addresses saved as regular bookmarks) for Brief see them
