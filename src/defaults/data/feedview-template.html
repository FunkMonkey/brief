Navigation
K or Space Select next entry
J or Backspace Select previous entry
Alt + K Show next page
Alt + J Show previous page
Enter Open entry
Ctrl + Enter Open entry in new tab
N Unfold or fold selected entry (when showing only headlines)
Entries
M Mark selected entry as read or unread
B Star or unstar selected entry
T Move selected entry to Trash
R Restore selected entry (when in Trash)
Ctrl + M Mark all viewed entries as read
Tab Deselect any entries
View
F Toggle headlines mode
A View all entries
S View starred entries
D View unread entries
/ Focus searchbox
? Display list of shortcuts keys
